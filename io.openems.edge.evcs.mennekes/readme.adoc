= Mennekes BRx and BBx Station

This component implements the Mennekes BRx and BBx charging station, which is controlled and read out using the proprietary TCP protocol.
It collects all relevant information into the given Nature Channels and its own Channels and sends charging commands that have been set by another controllers.

Implemented Natures:
* Evcs (Electric Vehicle Charging Station)
* ManagedEvcs
